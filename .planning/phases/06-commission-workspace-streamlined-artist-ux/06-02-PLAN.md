# Plan 06-02: Redesign Sidebar Controls into a 4-Step Wizard Workflow

Goal: Replace the Left Sidebar Tab system with a structured 4-step wizard journey guiding the artist sequentially from upload to checkout.

## Proposed Changes

### Component Area: Sidebar Navigation Wizard

#### [MODIFY] [App.tsx](file:///C:/Users/rickf/.gemini/antigravity/scratch/gempixel/src/App.tsx)
- Replace `activeTab` tab state with `wizardStep: number` state (defaulting to 1).
- Render a visual progress step track at the top of the Left Sidebar:
  - Consists of steps: 1 (Upload), 2 (Size & Style), 3 (Legend & Palette), 4 (Quoting & Checkout).
  - Renders glowing dots for completed/active steps, and connector lines indicating status.
- Group sidebar control components into four distinct panel views:
  - **Panel 1 (Ingestion):** File drop-zone, aspect ratio fit/crop options, and recent uploads gallery list.
  - **Panel 2 (Size):** Grid presets selection dropdown, manual columns/rows input, physical size outputs, and round/square styling toggles.
  - **Panel 3 (Palette):** Base kits selector, legend supply list table, highlighting controls, and exclusion indicators.
  - **Panel 4 (Quote):** Pricing database config table, safety margin select, cost estimates card, canvas custom redirects, and DiamondDrillsUSA checkout permalink CTAs.
- Render footer navigation buttons at the bottom of the sidebar panel:
  - Render a `← Back` button if `wizardStep > 1` (decrements `wizardStep`).
  - Render `Next: [Step Name] →` button if `wizardStep < 4` (increments `wizardStep`).
  - Disable "Next" button on Step 1 if no image has been loaded yet, ensuring the user follows a logical linear path.
