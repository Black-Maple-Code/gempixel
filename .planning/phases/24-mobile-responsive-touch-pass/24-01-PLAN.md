<objective>
Reflow the already-built Atelier 4-step journey (Upload → Refine → Supplies → Order) into a single portrait column at ~300px using Tailwind v4 container queries, keeping every control inline, and delete the dead legacy drawer CSS that encodes the abolished "drawer" model. Delivers MOBILE-01.

Purpose: A phone user must see the same four screens as one scrollable portrait column with the canvas-first preview pinned on Refine — never a sidebar or drawer. This is a purely additive CSS/layout pass: base desktop classes stay untouched, responsiveness is added as descendant-only `@max-[640px]:*` overrides so desktop is provably unregressed.
Output: Container-query reflow on AtelierShell + App shell body + RefineScreen, a sticky canvas-first Refine pane, and a cleaned `index.css` (dead drawer rules removed, live `.viewport-hud` mobile rule preserved).
</objective>
