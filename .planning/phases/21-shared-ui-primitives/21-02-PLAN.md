## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user gesture → native input value | Keyboard/pointer drives the native `<input type=range>`; the value is bounded by `min`/`max`/`step` and parsed with `parseInt`. No other external input. |
| consumer (App.tsx) → primitive props | First-party trusted; `options`/`label`/`value` flow in as controlled props. |

Stateless, render-only interactive primitives. No network, storage, auth, routing, or data mutation.
