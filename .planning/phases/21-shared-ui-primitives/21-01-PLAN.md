<objective>
Create the `src/ui/` directory and its foundation: the zero-dependency `cn()` class-join helper (D-02) plus the two simple variant-map primitives — `Button` (green `primary` CTA, dark `save` pill, bordered `ghost`) and `Pill` (`neutral` / `ok` / `tag` chips) — each pure/props-only, styled entirely from existing Atelier tokens, with co-located render tests.

Purpose: Give Phase 23 a consistent, token-driven Button/Pill layer so the four screens compose without duplicating Tailwind class soup (SC1).
Output: `src/ui/cn.ts`, `src/ui/Button.tsx`, `src/ui/Pill.tsx`, and their co-located tests under `src/ui/__tests__/`.
</objective>
